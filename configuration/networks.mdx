CoW Protocol Solver Rewards supports ten EVM networks. Set the `NETWORK` environment variable to the identifier in the table below to target a specific chain.

<Note>
  COW token rewards are only paid on networks where the real COW token is deployed (Mainnet, Gnosis, Arbitrum One, and Base). All other networks use a dummy address (`0x0000000000000000000000000000000000000006`) for the COW token, meaning COW reward amounts are computed but settled exclusively via the mainnet Gnosis Safe.
</Note>

## Network overview

| Network | `NETWORK` value | Dune blockchain name | DB name |
|---|---|---|---|
| Ethereum mainnet | `mainnet` | `ethereum` | `mainnet` |
| Gnosis Chain | `gnosis` | `gnosis` | `xdai` |
| Arbitrum One | `arbitrum` | `arbitrum` | `arbitrum-one` |
| Base | `base` | `base` | `base` |
| Avalanche C-Chain | `avalanche` | `avalanche_c` | `avalanche` |
| Polygon | `polygon` | `polygon` | `polygon` |
| BNB Smart Chain | `bnb` | `bnb` | `bnb` |
| Linea | `linea` | `linea` | `linea` |
| Plasma | `plasma` | `plasma` | `plasma` |
| Ink | `ink` | `ink` | `ink` |

## COW token support by network

COW token rewards are settled differently depending on whether a network has a real COW token deployment.

| Category | Networks |
|---|---|
| Real COW token | Mainnet, Gnosis Chain, Arbitrum One, Base |
| Dummy COW address (mainnet settlement) | Avalanche, Polygon, BNB, Linea, Plasma, Ink |

For networks using a dummy address, the script computes COW reward amounts in the same way as other networks but the actual token transfer is executed on Ethereum mainnet using `PAYOUTS_SAFE_ADDRESS_MAINNET`.
